
# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

This directory is the **frontend package** of a two-package monorepo. See the root `../CLAUDE.md` for the full architecture, API routes, auth flow, and dev commands.

## Commands (frontend only)

```bash
npm install    # Install frontend dependencies
npm run dev    # Vite dev server at http://localhost:3000
npm run build  # Build to build/ directory
```

Run `npm run dev:all` from the **root** to start both frontend and backend together.
